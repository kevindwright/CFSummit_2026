# What does Filter() do?

- Allows you to take an array and return a new array containing only the elements that meet a specific conditions.
- Legacy loops focus on iteration. filter() focuses on intent.
- Instead of describing the mechanics of iteration, you're describing the result you're after.
- What you want, not how to do it
- This leads to cleaner, more declarative code.

# Key considerations

- Reduces opportunities for bugs and makes maintenance easier.
- Original collection is NEVER modified — filter() returns a new collection
