# What does Reduce() do?

- Data Aggregation into a single result.
- reduce() accumulates a value as it processes each item in the collection.
- Concise - your code communicates intent instead of mechanics, it becomes easier to read, easier to maintain, and much harder to break.

# Key considerations

- Original collection is NEVER modified — reduce() returns a new collection
- That result could be a number, a string, a structure, an array, or even a complex object.
